# 🚀 Smart Queue System (.NET 8)

A modern queue management system built with ASP.NET Core.

## 📌 Features
- Take ticket online
- Real-time queue tracking
- SMS notifications
- Admin dashboard
- REST API ready for integration

## 🧠 Use Cases
- Hospitals
- Government services
- Telecom companies
- Clinics

## 🛠️ Tech Stack
- .NET 8 Web API
- Clean Architecture
- SQL Server
